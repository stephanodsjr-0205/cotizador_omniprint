# Agregar Logo al PDF

Para agregar un logo personalizado a los PDFs exportados:

1. Coloca tu archivo de logo en la carpeta `public/` con el nombre `logo.png`
2. El logo debe ser en formato PNG
3. Tamaño recomendado: 200x80 píxeles (o proporción similar)
4. El logo se mostrará automáticamente en la parte superior izquierda del PDF

Si no hay logo, se mostrará el texto "COTIZADOR OMNI" como encabezado.

## Ubicación del archivo:
```
public/
  └── logo.png
```

## Formatos soportados:
- PNG (recomendado)
- JPG/JPEG
- SVG (puede requerir conversión)
